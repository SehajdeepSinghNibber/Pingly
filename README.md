# 💬 Pingly — Real-Time Chat Application

A modern real-time messaging platform built with the **MERN Stack**, featuring secure authentication, instant messaging, online presence tracking, and a responsive user experience.

## 🚀 Features

### 🔐 Authentication & Security

* User signup and login
* JWT-based authentication and authorization
* Secure password hashing with bcrypt
* Protected routes and API endpoints

### 💬 Real-Time Messaging

* Instant one-to-one messaging using Socket.io
* Real-time message delivery
* Live typing and communication experience
* Message persistence with MongoDB

### 🟢 Online Presence

* Real-time online/offline user status
* Active user tracking using Socket.io
* Dynamic updates without page refresh

### 🎨 Modern UI/UX

* Responsive design for desktop and mobile
* Built with Tailwind CSS and DaisyUI
* Clean and intuitive chat interface
* Theme customization support

### ⚡ State Management

* Global state management with Zustand
* Efficient and scalable application architecture
* Optimized client-side performance

### 🛠️ Error Handling

* Robust server-side validation
* Client-side error management
* User-friendly notifications and feedback

### 🌐 Deployment Ready

* Production-ready configuration
* Free deployment support
* Environment-based configuration management

## 📂 Environment Variables

Create a `.env` file in the root directory and add the following:

```env
PORT=5000
MONGO_DB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
NODE_ENV=development
```

## 📦 Installation

### Clone the Repository

```bash
git clone https://github.com/SehajdeepSinghNibber/Pingly.git
cd Pingly
```

### Install Dependencies

```bash
npm install
```

---

## ▶️ Run Development Server

```bash
npm run dev
```

The application will start in development mode.

---

## 🏗️ Build for Production

```bash
npm run build
```

---

## 🚀 Start Production Server

```bash
npm start
```
